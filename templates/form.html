Insight & Analisis Data

  • Distribusi kelas “result” tidak seimbang, kelas 0 jauh lebih banyak dari kelas 1.
  • MOI menjadi faktor paling signifikan membedakan kebutuhan irigasi.
  • Tidak ada missing value, dataset bersih dan siap digunakan.
  • Fitur kategorikal soil_type dan Seedling Stage relevan untuk prediksi.
  • Korelasi paling kuat antara MOI dan label “result”.
  • Dataset cocok untuk model klasifikasi kebutuhan irigasi.

Metodologi Singkat

  1. EDA: Statistik dasar, distribusi fitur, korelasi antar variabel.
  2. Pra-pemrosesan: Label encoding, scaling, feature selection.
  3. Pemodelan: Decision Tree, Random Forest, KNN.
  4. Evaluasi: Akurasi, confusion matrix, feature importance.
  5. Deployment: Form prediksi & dashboard visualisasi.

Kesimpulan

Tujuan proyek tercapai dengan baik: model machine learning mampu memprediksi kebutuhan irigasi tanaman secara akurat berdasarkan data sensor pertanian. Model XGBoost dan Random Forest menunjukkan performa sangat tinggi, dengan akurasi mendekati sempurna pada data uji. Fitur seperti MOI, soil_type, dan Seedling Stage terbukti sangat berpengaruh dalam prediksi.

Untuk pengembangan lebih lanjut, disarankan melakukan evaluasi pada data eksternal untuk memastikan generalisasi model, serta menambah fitur lokasi dan waktu agar prediksi lebih kontekstual. Integrasi dengan sistem monitoring real-time dan rekomendasi tindakan otomatis dapat meningkatkan manfaat bagi petani dan stakeholder pertanian.
